Gecko-Weather

A simple weather widget for Geckoboard

Instructions

  1. Find your location's WOEID: WOEID Lookup
  2. Enter it here:
  3. Choose your preferred temperature format:
  4. Copy this URL:
  5. Create a new custom text widget on your Geckoboard
  6. Paste the URL into the URL Data Feed field
  7. Set the Widget Type to Custom
  8. Set the Feed Format to JSON
  9. Add the widget to your dashboard

Details

This app uses the Yahoo Weather API to fetch current forecast info for your location then converts it into a format that works with Geckoboard's custom text widget.

Built by David Underwood in about 4 hours (including research).
